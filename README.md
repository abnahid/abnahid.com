# Foobar

Foobar is a HTML CSS JS library for dealing with word pluralization.

## Vew site

Vew Site link [ab.nahid](https://abnahid.com) to Vew Now .

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
